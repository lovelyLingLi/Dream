# Dream UI

Dream个人博客系统前端界面，基于Vue 3 + Vite构建的现代化Web应用。

## 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 下一代前端构建工具
- **Vue Router** - Vue.js官方路由管理器
- **Tailwind CSS** - 实用优先的CSS框架
- **Heroicons** - 精美的SVG图标库

## 开发指南

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```
